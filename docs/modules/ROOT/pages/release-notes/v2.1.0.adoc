= v2.1.0

== Changes / updates

* The response context includes the original requestId when it is applicable
* Documentation fixes regarding OPENTELEMETRY_SAMPLER_TYPE and OPENTELEMETRY_SAMPLER_RATIO environment variables
* The `dookug.service.interface.parametersdata.gzipped` config default value changed to `true`,
and with the true value the backend can process compressed and uncompressed parameters too.

== Migration

Changes are backwards compatible, doesn't need any migration.
