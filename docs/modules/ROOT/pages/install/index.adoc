= Installation, Deployment

The Dookug module must be accessible in the environments of the project(s) that intend to utilize the service. 
Each instance of the service, along with its infrastructure requirements, should be installed and configured for every environment (development/test/production).

This includes:

* a relational database (Oracle or Postgres)
